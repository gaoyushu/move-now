## 项目简介
----
Move Now是一款学习+社交的健身APP。
它内含丰富的健身教学视频，你可以选择课程，制定个性化的健身计划，记录你的健身动态。
同时，你可以看到别人的健身动态，认识健身好友，相互沟通交流，分享健身心得体会和健身经验。
